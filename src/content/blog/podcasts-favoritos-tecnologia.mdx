Como entusiasta de tecnologia, desenvolvimento web e JavaScript, meu meio principal de ficar por dentro das novidades é através de podcasts. Neste artigo, compartilho alguns dos meus podcasts favoritos que me ajudam a aprender sobre os novos frameworks de frontend, novidades em CSS, soluções serverless e muito mais.


### [React Native Radio](https://open.spotify.com/show/28hPRLml3FbP14FCYtlOvg?si=f0cd1010da8c4475)

Este podcast é apresentado pela Infinite Red, uma empresa de desenvolvimento mobile americana especializada em React Native. Eu recomendo começar pelo episódio com a Coinbase, onde eles discutem a transição de desenvolvimento nativo para React Native.

- [Transitioning from Native to React Native at Coinbase](https://open.spotify.com/episode/7z4ejJOUouGYNpMFYQVuO4)

### [COMPRESSEDfm](https://open.spotify.com/show/7Er1m8VEzRNu59ZhaS18WY)

Este é um excelente podcast sobre design e desenvolvimento web. Eu adoro a apresentadora Amy Dutton e as entrevistas com vários convidados interessantes. Para começar, recomendo o episódio sobre o framework Astro, que eu usei para criar este blog.

- [Making the Web Weird Again with Astro](https://open.spotify.com/episode/0StBL1uw96DevWOHOfOaVA)

### [The Changelog](https://open.spotify.com/show/5bBki72YeKSLUqyD94qsuJ)

Este podcast apresenta os melhores hackers, líderes e inovadores do mundo do software e open source. As conversas são longas, mas sempre super interessantes, com duração de mais de uma hora. Recomendo muito o episódio com Adam Wiggins, co-fundador e ex-CTO do Heroku.

- [The Story of Heroku](https://open.spotify.com/episode/3nEwB6apkvUpwk8W9KgzEF)

### [Syntax](https://open.spotify.com/show/4kYCRYJ3yK5DQbP5tbfZby)

Este é o podcast mais famoso da lista, aparecendo em primeiro lugar no State of JS 2022.  Muito focado em desenvolvimento Web, novidades de CSS, Typescript, browsers, etc. Gostei do episódio de halloween com histórias divertidas e assustadoras de desenvolvedores.

- [Spooky Web Dev Stories 2022](https://open.spotify.com/episode/1ayyziSncWIOL0OthqiDvA)

### [PodRocket](https://open.spotify.com/show/6oFuKu89C9X1wQ7bT0QEM2)
Outro podcast que sempre traz as novidades do mundo do desenvolvimento web. Gostei bastante do episódio com o criado do Zod que é uma biblioteca muito boa para trabalhar com validação de dados com Typescript.

- [Zod with Matt Pocock](https://open.spotify.com/episode/6M0EsRlPoQa4OmBNh2CV8e)

### [ShopTalk](https://open.spotify.com/show/2PUoQB330ft0sTzSNoCPrH)

Muitas vezes eu pulo os episódios desse podcast, mas algumas semanas tem algum convidado interessante. Gostei muito do episódio com time da PlanetScale.

- [PlanetScale with Iheanyi Ekechukwu and Mike Coutermarsh](https://open.spotify.com/episode/2SLMMKBJQVKEJWxLmQ4aIF)
